## Getting Started

* Download and install [VLC media player](https://www.videolan.org/vlc/index.html)

* Navigate to the `gui` directory

### macOS

```Bash
conda env create -f environment.yml
source activate data_vis_env
pythonw wxvlc.py
```

### Linux

```Bash
conda env create -f environment.yml
source activate data_vis_env
python wxvlc.py
```

### Windows

```Bash
conda env create -f environment.yml
activate data_vis_env
python wxvlc.py
```
